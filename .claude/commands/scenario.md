Запуск сценариста для серии.

Аргумент: $ARGUMENTS (название истории или свободная тема)

Шаги:
1. Прочитай SERIES_BIBLE.md — персонажи, тон, формат
2. Прочитай ~/Documents/WORK/AI_OFFICE/refs/MY_STORIES.md — банк историй
3. Найди указанную историю: "$ARGUMENTS"
4. Если история не найдена — скажи и предложи список доступных историй
5. Напиши раскадровку строго по формату из SERIES_BIBLE.md:
   - 8-12 кадров
   - Каждый кадр: ВИЗУАЛ + ГОЛОС + НАСТРОЕНИЕ + MIDJOURNEY-промпт
   - Зацепка с первой секунды
   - Один инсайт
   - Юмор обязателен
   - Финал — вопрос или крючок
6. Подставь URL мастер-образов из assets/characters/PROMPTS.md в --cref/--sref (если заполнены)
7. Добавь цельный текст озвучки и заметки для продакшена
8. Покажи раскадровку на утверждение. После утверждения — сохрани в scripts/S01EXX_ru.md

КРИТИЧЕСКОЕ ПРАВИЛО: Основа истории — реальная (из MY_STORIES.md). Подача — творческая. НЕ выдумывать факты, цифры, ситуации.
