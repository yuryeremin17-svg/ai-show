# FILMING_GUIDE.md -- Инструкция по съёмке для Юрия

> Как снять себя на iPhone за 10 минут. Без оператора, без студии, без актёрского опыта.
> Формат: 2 кадра по 10 секунд (зацепка + инсайт). Вертикальное видео 9:16.

---

## 1. ЧЕКЛИСТ ПЕРЕД СЪЁМКОЙ (5 минут)

### Оборудование -- собрать один раз, потом не трогать
- [ ] iPhone (основная камера, НЕ селфи-камера -- качество выше)
- [ ] Штатив-тренога с держателем для телефона (высота -- уровень глаз стоя или сидя)
- [ ] Беспроводной петличный микрофон (DJI Mic Mini, RODE Wireless Micro или Hollyland Lark M2)
- [ ] Белая тюль / рассеиватель на окне (если солнце яркое)
- [ ] Зеркало ИЛИ ноутбук с зеркальным экраном (для контроля кадра при съёмке на заднюю камеру)

### Помещение
- [ ] Тихая комната (выключить кондиционер, закрыть окна, убрать телефоны из зоны)
- [ ] Окно впереди-сбоку (свет падает на лицо, НЕ за спиной)
- [ ] Фон: чистый, минимальный -- книжная полка, растение, одноцветная стена. Без хлама

### Внешний вид
- [ ] Однотонная одежда (тёмно-синий, серый, чёрный). Без полосок, мелкого рисунка, блестящего
- [ ] Причёска в порядке, ничего не свисает на лицо
- [ ] Если лицо блестит -- промокнуть матирующей салфеткой

### Камера (настроить один раз)
- [ ] Режим видео, 4K 30fps (или 1080p 60fps)
- [ ] Телефон вертикально (портрет, 9:16)
- [ ] Задняя камера (главная, НЕ селфи)
- [ ] Включить сетку 3x3 в настройках камеры (Настройки → Камера → Сетка)
- [ ] Зафиксировать экспозицию: долгое нажатие на лицо → появится "Фиксация АЭ/АФ"
- [ ] Пробный кадр 5 секунд → посмотреть: свет, звук, кадрирование

## 2. КАДРИРОВАНИЕ (вертикальное 9:16)

### Золотые правила
```
┌─────────────────────┐
│  ▒▒▒ UI зона ▒▒▒   │ ← Верхние 10-15%: ник, аватарка платформы
│                     │
│     ┌─────────┐     │
│     │  ГЛАЗА  │     │ ← Глаза на 30-35% от верха кадра
│     │  здесь  │     │   (на верхней линии сетки 3x3)
│     │         │     │
│     │  грудь  │     │ ← Кадр от середины груди и выше
│     └─────────┘     │
│                     │
│  ▒▒ кнопки ▒▒▒▒▒   │ ← Правый край 10-15%: лайки, комменты
│  ▒▒▒ подписи ▒▒▒   │ ← Нижние 15-25%: подпись, хештеги
└─────────────────────┘
```

### Конкретные числа (для 1080x1920)
| Параметр | Правило | Пиксели |
|----------|---------|---------|
| Глаза | 30-35% от верхнего края | 576-672 px от верха |
| Зазор над головой | 8-12% | 150-230 px |
| Зона для текста | только 20-60% от верха | --- |
| Мёртвая зона внизу | 60-100% от верха | не ставить текст |

### Как проверить кадрирование
1. Включить сетку 3x3 в настройках камеры
2. Глаза -- на верхней горизонтальной линии сетки
3. Над головой -- зазор примерно в кулак (8-12%)
4. Видно от середины груди и выше
5. Тело по центру, не прижат к краю

### Куда смотреть
- **ПРЯМО В ОБЪЕКТИВ задней камеры.** Не на экран. Не "примерно туда". Точно в линзу.
- Наклеить маленькую цветную точку (стикер) рядом с объективом -- ориентир для взгляда
- Если используешь телепромптер на втором устройстве -- расположить его вплотную к камере

## 3. СВЕТ (Дубай = много солнца, это подарок)

### Схема "Одно окно" (лучшая для домашней съёмки)

```
         [ОКНО с тюлью]
              │
              │  свет
              │
              ▼
    ┌─────────────────┐
    │     ЮРИЙ        │  ← Лицом к окну, слегка под углом 30-45°
    │     (стул)      │
    └─────────────────┘
              │
              │
         [КАМЕРА на штативе]
```

### Правила
1. **Свет = спереди.** Окно за камерой или чуть сбоку. НИКОГДА за спиной (силуэт)
2. **Рассеивать прямое солнце.** Белая тюль / штора / бумага на окне. Без рассеивателя -- жёсткие тени
3. **Лучшее время в Дубае:** 7:00-9:00 утра или 16:00-17:30 (мягкий свет, золотой час)
4. **Полдень:** солнце сверху, тени под глазами. Закрыть шторы, использовать кольцевой свет
5. **Нет окна?** Кольцевой свет (ring light) 10-12 дюймов за камерой, на уровне глаз
6. **Заполняющий свет (опционально):** белый лист бумаги А3 напротив окна -- отражает свет на теневую сторону лица

### Чего избегать
- Потолочный свет (тени под глазами, "пещерный" эффект)
- Окно прямо за спиной (лицо тёмное, фон пересвечен)
- Смешанный свет (солнце + лампа с жёлтым светом = разные цвета кожи)

## 4. ЗВУК

### Приоритет (от лучшего к худшему)
1. **Беспроводная петличка** (DJI Mic Mini ~$170, RODE Wireless Micro ~$80) -- прицепить к воротнику, 15 см от рта
2. **Проводная петличка** (RODE smartLav+ ~$60) -- дешевле, но провод мешает
3. **Встроенный микрофон iPhone** -- только если больше ничего нет. Максимум 50 см от рта

### Правила звука
- Кондиционер ВЫКЛЮЧИТЬ (даже тихий -- микрофон слышит)
- Закрыть окна и двери
- Положить подушку / полотенце за камерой (гасит эхо в пустой комнате)
- Записать 10 секунд тишины перед съёмкой → послушать → если есть гул -- найти источник
- Говорить чётко, не мямлить, но НЕ кричать

### Проверка звука
Записать тестовые 10 секунд. Послушать в наушниках. Если слышно:
- Шипение → микрофон слишком далеко или усиление слишком высокое
- Гулкость → эхо комнаты, добавить мягкие поверхности
- Щелчки → микрофон трётся об одежду, закрепить

## 5. КАК ДЕРЖАТЬСЯ ПЕРЕД КАМЕРОЙ (для неактёра)

### Главное правило
**Ты не выступаешь. Ты рассказываешь другу.** Представь, что за камерой сидит конкретный человек -- тот, кому ты реально это рассказал бы.

### Тело
- **Стоя** (предпочтительно): ноги на ширине плеч, вес на обе ноги, чуть наклон вперёд
- **Сидя** (если удобнее): на краю стула, спина прямая, НЕ откинувшись назад
- **Плечи:** расправлены, опущены. НЕ подняты к ушам
- **Руки:** видны в кадре, на уровне груди. Жестикулировать МОЖНО и НУЖНО -- это выглядит живо
- **Запрещено:** руки в карманах, скрещены на груди, теребить что-то, чесать лицо, трогать волосы

### Лицо и глаза
- Смотреть ПРЯМО В ОБЪЕКТИВ (не на экран, не в сторону)
- Лёгкая улыбка по умолчанию -- не натужная, не "сыр"
- Брови чуть приподняты -- добавляет энергии
- Моргать естественно. НЕ таращиться

### Энергия
- **Правило +20%:** Камера съедает энергию. Если в жизни говоришь на 7/10 -- на камере будет 5/10. Добавь 20% к своему обычному уровню
- Перед записью: 5 глубоких вдохов, расправить плечи, встряхнуть руки
- Начинать фразу на выдохе (голос звучит ниже и увереннее)

### Голос
- Темп: чуть медленнее чем кажется нормальным. 140-160 слов/минуту
- Паузы ВМЕСТО слов-паразитов ("э-э", "ну", "как бы"). Пауза = сила. "Э-э" = слабость
- Ключевые слова -- выделять голосом (громче / медленнее / после паузы)
- Финальную фразу -- чуть тише и медленнее (создаёт вес)

### Разминка перед съёмкой (2 минуты)
1. Расправить плечи, потянуться вверх, опустить -- 3 раза
2. Покрутить головой -- 3 раза в каждую сторону
3. Открыть рот максимально широко, закрыть -- 5 раз (расслабляет челюсть)
4. Произнести вслух свой текст 2 раза -- НЕ в камеру, просто вслух, как будто рассказываешь
5. Улыбнуться, глубокий вдох -- включить запись

## 6. ПОДАЧА ТЕКСТА: телепромптер vs. запоминание vs. тезисы

### Для формата 10 секунд -- рекомендация: ТЕЗИСЫ + НЕСКОЛЬКО ДУБЛЕЙ

| Метод | Плюсы | Минусы | Для кого |
|-------|-------|--------|----------|
| **Тезисы (2-3 слова)** | Естественная подача, живой взгляд | Нужно 3-5 дублей | **Рекомендуется для Юрия** |
| Телепромптер | Точный текст, меньше дублей | Бегающие глаза, роботный голос | Опытные ведущие, длинные тексты |
| Наизусть | Самый естественный результат | Стресс, "замерзание", 10+ дублей | Профи, актёры |

### Как работать с тезисами
1. Написать полный текст (25-30 слов на 10 секунд)
2. Выписать 3-5 ключевых слов КРУПНО на листе бумаги
3. Закрепить лист РЯДОМ с камерой (но не читать -- только глянуть перед записью)
4. Рассказать своими словами, глядя в объектив
5. Не гнаться за точностью -- важны смысл и энергия

### Если всё-таки телепромптер
- **Приложение:** BIGVU (бесплатная версия) или Teleprompter.com
- Второй телефон/iPad вплотную к камере (текст рядом с объективом)
- Скорость: 130-140 слов/мин (медленнее чем кажется)
- Размер шрифта: МАКСИМАЛЬНЫЙ
- Предложения: короткие, 5-8 слов. Разговорный стиль, сокращения
- Прочитать вслух 2 раза ДО записи -- чтобы привыкнуть к ритму
- ПРОБЛЕМА: при 10 секундах телепромптер избыточен -- тезисы быстрее

## 7. СТРАТЕГИЯ ДУБЛЕЙ

### Сколько снимать
- **Минимум:** 5 дублей каждого кадра (зацепка и инсайт отдельно)
- **Реально:** 7-10 дублей. Это нормально. Даже профи делают 5+
- **Когда остановиться:** когда есть 2 хороших дубля подряд

### Какой дубль выбрать
- **Обычно лучший -- дубль 3-5.** Первые 2 -- разогрев. После 7 -- усталость
- Выбирать по ЭНЕРГИИ, не по точности слов. Живой > правильный
- Если запнулся на одном слове -- не останавливай запись, продолжай. Вырежешь потом

### Процесс съёмки одного кадра (10 секунд)
1. Встать в позицию, посмотреть на тезисы
2. Глубокий вдох
3. Нажать запись
4. Подождать 2 секунды (для монтажа)
5. Сказать текст, глядя в объектив
6. Подождать 2 секунды после последнего слова
7. Остановить запись
8. НЕ СМОТРЕТЬ сразу -- снять ещё 2-3 дубля подряд, потом смотреть

### Ловушка перфекциониста
- После 10 дублей -- СТОП. Взять лучший из того что есть
- Идеала не будет. "Достаточно хорошо" = хорошо. Аудитория ценит живость, а не идеальность
- Напоминание: это 10 секунд из 60-секундного видео. Остальные 50 -- анимация

## 8. ТИПИЧНЫЕ ОШИБКИ НЕАКТЁРОВ

| Ошибка | Почему плохо | Как исправить |
|--------|-------------|---------------|
| Смотрит на экран, а не в объектив | Взгляд "мимо" зрителя, нет контакта | Наклеить стикер рядом с линзой |
| Говорит монотонно | Зритель уходит через 3 секунды | Правило +20% энергии. Выделять ключевые слова |
| Читает текст | Видно что читает -- глаза бегают, голос неестественный | Тезисы вместо полного текста |
| Руки висят / скрещены | Закрытая поза, напряжение | Жестикулировать. Руки на уровне груди |
| Трогает лицо / волосы | Отвлекает зрителя от текста | Осознать привычку. Снять 1 дубль на проверку |
| Стоит на месте как памятник | Неживая картинка | Лёгкий наклон вперёд, жесты, движение головой |
| Говорит "э-э-э", "ну", "значит" | Убивает авторитет | Паузы вместо заполнителей. Тренировать |
| Фон -- бардак | Зритель смотрит на фон, не на тебя | 30 секунд уборки перед съёмкой |
| Свет сзади (окно за спиной) | Тёмное лицо, пересвеченный фон | Развернуться лицом к окну |
| Слишком близко / далеко | Некомфортная дистанция для зрителя | Кадр от середины груди. Голова не обрезана |
| Спешит, глотает слова | Текст не усваивается | Темп 140-160 сл/мин. Пауза перед ключевой фразой |
| Пересматривает каждый дубль | Теряет энергию, начинает себя критиковать | Снять 3 дубля подряд, потом смотреть |

## 10. ПОШАГОВЫЙ ПРОЦЕСС СЪЁМКИ (10 минут на всё)

### Подготовка (5 мин, первый раз. Потом 2 мин)
1. Поставить штатив у стены напротив окна
2. Закрепить iPhone вертикально, объектив на уровне глаз
3. Включить сетку, включить видео 4K/30fps
4. Прицепить петличку к воротнику
5. Включить фиксацию экспозиции (долгое нажатие на лицо)
6. Тестовый кадр 10 секунд → проверить свет, звук, кадрирование
7. Разминка 2 минуты (п.5 выше)

### Съёмка кадра "Зацепка" (3 мин)
1. Прочитать тезисы (3 ключевых слова)
2. Убрать бумагу с тезисами из кадра
3. Вдох → запись → 2 сек тишины → текст → 2 сек тишины → стоп
4. Повторить 5-7 раз без просмотра
5. Посмотреть дубли, выбрать лучший по энергии

### Съёмка кадра "Инсайт" (3 мин)
1. Те же шаги что выше
2. Интонация: спокойнее, весомее, чуть тише. Это вывод, не зацепка
3. Если есть вопрос к зрителю -- смотреть прямо в объектив, пауза после вопроса

### После съёмки (1 мин)
1. Скинуть видео в `assets/scenes/<episode>/` (или AirDrop на Mac)
2. Назвать файлы: `yuri_hook.mov` и `yuri_insight.mov`
3. Готово. Остальное делает assemble.py
