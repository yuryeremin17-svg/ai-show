# PIPELINE — AI Office Studio

> Производственный конвейер: 9 ролей, voice-first.
> Один эпизод = один проход конвейера.
> Claude надевает шапки последовательно.
> Юрий — босс: тема, съёмка, утверждение.

---

## ПРИНЦИП: ГОЛОС — МЕТРОНОМ

Голос записывается ДО генерации картинок.
Голос фиксирует хронометраж каждого шота.
Аниматик проверяет ритм ДО дорогой генерации.
Картинки подгоняются под голос, не наоборот.

## ЧТО ДЕЛАЕТ ЮРИЙ НА КАЖДОМ ШАГЕ

| Шаг | Действие Юрия | Время |
|-----|---------------|-------|
| 1. Сценарист | Даёт тему. "Да / нет / поменяй" | 5 мин |
| 2. Раскадровщик | Смотрит shot list. "Понятно / вопрос" | 5 мин |
| 3. Voice director | Выбирает голос агента из 2-3 вариантов | 5 мин |
| 4. Аниматик | Смотрит черновик. "Ритм ок / шот Х длинный" | 5 мин |
| 5. Промпты + генерация | Вставляет промпт в MJ, выбирает из 4-х. Загружает в Seedance. Снимает 2 клипа | 60 мин |
| 6. Sound designer | Генерирует трек в Suno по промпту. "Годится?" | 10 мин |
| 7. Монтаж | — (Claude делает) | 0 мин |
| 8. QC | Показывает ролик эксперту | 10 мин |
| 9. Локализация | Снимает 2 клипа на англ. (опц.) | 15 мин |
| **Итого** | | **~2 часа** |

## ПОРЯДОК СБОРКИ СТУДИИ

Не всё сразу. Строим по одному, проверяя на S01E01:

```
Неделя 1 (текущая):
  [x] Сценарий S01E01 — ГОТОВ
  [x] Раскадровка S01E01 — ГОТОВ (v2, два мира)
  [ ] voice_gen.py — нужен API ключ ElevenLabs
  [ ] Тест: голос Юрия + голос Клодища

Неделя 2:
  [ ] animatic.py — грубый черновик S01E01
  [ ] Юрий утверждает ритм
  [ ] ART_DIRECTION.md — стайл-гайд
  [ ] Юрий генерирует картинки + видео по промптам
  [ ] Юрий снимает 2 клипа

Неделя 3:
  [ ] audio_mix.py — аудио-микс
  [ ] assemble.py v3 — финальная сборка
  [ ] Первый готовый ролик → QC
  [ ] Английская версия

После первого эпизода:
  [ ] /storyboard — формализовать в скилл
  [ ] Batch: S01E02, S01E03
  [ ] Цель: 2-3 эпизода в неделю
```
