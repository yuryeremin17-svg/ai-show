## ПЛАН НА СЛЕДУЮЩУЮ СЕССИЮ

### Юрий делает сам (~2.5 часа):
1. **Послушать голоса** (5 мин) — `assets/voice/S01E02/`, файлы 01→22
2. **Midjourney — 7 картинок** (30 мин) — промпты из `scripts/S01E02_ru.md`
   - intro (дверь), пустой офис, Клодище, Тоня, вместе, офис-финал, outro (дверь)
3. **Seedance / Dreamina — ~14 клипов** (40 мин) — промпты из раскадровки
4. **Suno — джингл + фоновая музыка** (10 мин)
5. **Сборка в CapCut** (60 мин) — по порядку: интро → голоса → аутро + субтитры

### Claude делает по запросу:
- Перегенерация отдельных фраз: `voice_gen.py S01E02 --skip-existing` (2 мин)
- Правки текста в раскадровке
- Новые серии через /scenario
